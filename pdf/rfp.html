🔮 AI 기반 스마트 기능

문서 작업의 패러다임을 바꾸는 지능형 기능들입니다. 유료/무료 사용자에 따라 최적화된 AI 모델을 제공합니다.

  • 📑문서 요약: AI가 PDF 전체 내용을 핵심 문장으로 자동 요약
  • 🌐전체 번역: 문서를 영어/일본어/대만어 등으로 통번역하여 새 창으로 제공
  • 🖱️스마트 드래그: 드래그 시 번역, 단어 설명, 요약, OCR 기능 툴팁 표시
  • 👁️OCR 텍스트 인식: 이미지 기반 PDF에서도 텍스트를 인식하여 복사/검색
  • 🤖AI 모델: OpenAI(유료), 로컬 탑재 AI(무료) 이원화 운영

📖 기본 문서 보기

빠르고 직관적인 문서 열람 환경을 제공하여 사용자의 가독성을 높입니다.

  • 📂문서 열람: 멀티 탭(PC)으로 여러 파일 동시 열람
  • 🔍탐색 및 검색: 썸네일/목차 보기 및 실시간 텍스트 하이라이팅 검색
  • 🔄화면 제어: 확대/축소(15% 단위), 페이지 맞춤, 90도 회전
  • 🌙다크 모드: 눈의 피로를 줄여주는 어두운 테마 지원
  • 🕒최근 문서: 최근 열람/수정한 문서 목록 바로가기

🎨 고급 문서 편집 및 페이지 조작

문서의 모든 요소를 자유롭게 제어하고 재구성할 수 있는 강력한 편집 도구를 제공합니다.

텍스트 편집

  • ✓ 글꼴, 크기, 색상, 스타일

도형 및 이미지

  • ✓ 테두리, 채우기, 선 종류

펜 & 강조

  • ✓ 펜/형광펜 굵기, 색상, 투명도
  • ✓ 지우개 기능
  • ✓ 실행취소/되돌리기

페이지 관리

  • ✓ 추가/삭제/분할/병합/추출
  • ✓ 순서 변경 (드래그앤드롭)
  • ✓ 페이지 회전 (짝/홀/모두)

페이지 번호

  • ✓ 내용, 글꼴, 크기, 색상, 위치
  • ✓ 적용 페이지 지정 가능
  • ✓ 실시간 뷰어 제공

워터마크

  • ✓ 텍스트/이미지 워터마크
  • ✓ 각도, 투명도, 패턴 형태
  • ✓ 실시간 뷰어 제공

🖋️ 전자 서명 및 보안

강력한 보안 기능으로 문서를 안전하게 보호하고, 간편한 전자 서명으로 계약 프로세스를 단축합니다.

  • ✍️다양한 서명 방식: 이미지 삽입, 텍스트 입력, 직접 그리기
  • 🕰️타임스탬프: 서명 시점의 시간 정보를 기록하여 법적 효력 강화
  • 🔒문서 잠금: 암호 설정으로 열람 제한
  • 🚫권한 제어: 복사, 인쇄, 편집 등 특정 기능 제한

🔄 파일 변환 및 내보내기

PDF를 중심으로 다양한 포맷과 자유롭게 상호 작용합니다.

  • ⇆상호 변환: PDF ↔ PPT, Word, Excel, JPG 등
  • 📦PDF 압축: 용량이 큰 PDF 파일을 최적화하여 저장
  • 🖨️인쇄: PC 환경에서 표준 프린터로 출력
  • 📲공유: 모바일에서 SNS, 메일, 메시지로 간편하게 공유

📋 폼(Form) 지원

문서 내 입력 양식을 자동으로 인식하여 데이터 입력을 편리하게 합니다.

  • 📝필드 자동 인식: 입력 가능한 필드를 자동으로 감지하여 활성화
  • ✔️다양한 입력 요소: 텍스트, 체크박스, 라디오, 드롭다운, 날짜 지원

👥 사용자 시스템

간편한 로그인부터 개인화된 마이페이지까지, 사용자 중심의 시스템을 구축합니다.

  • 🔗SNS 간편 로그인: 카카오, 네이버, 구글 등으로 빠른 시작
  • 👤마이페이지: 프로필 관리 및 최근 열람 문서 확인
