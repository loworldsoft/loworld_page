Loworld AI 창업자 이력

AI Agent로 미래를 이끌어갈 Loworld AI

개발자가 된 이유

2002년 해군 순항훈련 중 13개국을 방문하며 극심한 국가 간 빈부격차를 목격했습니다. 이 경험은 저에게 깊은 생각의 몰입을 안겨주었고, '달러를 벌어야 한다'는 결론에 이르게 했습니다. 온라인 공간에 빌딩을 세울 수 있는 개발자가 되어 이 목표를 이루겠다는 꿈을 품게 되었습니다.

이후 디자인 전공을 포기하고, 군 복무 중 영어 공부에 매진하며 개발자의 꿈을 향해 나아갔습니다. 제대 후에는 영국으로 건너가 본격적으로 개발자의 삶을 시작했습니다.

주요 경력

영국에서 .NET 기술을 습득한 후, 스위스 스포츠 영상 분석 소프트웨어 회사에서 로잔공대 박사들과 솔루션을 개발하며 공학적 지식과 객체지향 프레임워크 개발 역량을 키웠습니다. 이 경험은 저를 숙련된 개발자로 성장시키는 발판이 되었습니다.

주요 문제 해결 경험

해외 박물관에 전시된 PixelSense 메모리 누수 문제 해결

디스트릭트(d'strict)는 혁신적인 프로젝트 진행 중 발생하는 기술적 난관 해결을 위해 저에게 자문을 요청하곤 했습니다. 특히 기억에 남는 사례는 해외 고객사의 대규모 인터랙티브 전시 시스템에서 발생한 퍼포먼스 이슈였습니다. 현장에서 사용자 증가에 따라 시스템 프리징 현상이 보고되었지만, 해외 전시라는 특수한 상황으로 인해 직접적인 디버깅이 불가능했습니다. 이에 저는 원격으로 문제 해결에 착수했습니다. 우선 시스템 부하 테스트 환경을 구축하여 실제 운영 환경과 유사한 상황을 시뮬레이션하고, 메모리 프로파일링 툴을 활용하여 메모리 누수 발생 지점을 추적했습니다.

심층 분석 결과, WPF(PixelSense) 프레임워크 기반 UI 레이아웃에서 중첩 Grid 사용으로 인한 이벤트 버블링이 과도한 메모리 사용을 야기하고 있음을 발견했습니다. 이에 이벤트 핸들러 최적화 및 메모리 관리 전략 개선을 통해 문제를 해결하고, 원격 패치를 통해 시스템 안정성을 확보했습니다.

이러한 경험을 통해 저는 원격 환경에서의 문제 해결 능력과 복잡한 시스템 아키텍처에 대한 깊이 있는 이해를 바탕으로 실질적인 솔루션을 제공하는 전문성을 입증했습니다. (지금은 없어진 MS PixelSense)

실시간 억단위 빅데이터 병렬/분산 처리 및 Virtual Gantt Chart 개발 (줌/펜)

삼성 디스플레이의 고도화된 자동화 생산 라인은 높은 생산성을 자랑했지만, 수많은 지연 요소로 인해 병목 현상 분석에 어려움을 겪고 있었습니다. 이를 해결하기 위해 모든 공정 요소에 센서를 적용하여 실시간 데이터를 수집하는 시스템이 필요했습니다. 기존 SDS의 E-MAS 시스템은 빅데이터 분석에 20분 이상 소요되고 잦은 메모리 부족 오류로 인해 실시간 분석이 불가능했습니다. 이에 저는 대규모 데이터 처리 성능을 획기적으로 향상시킨 E-MAS2 시스템 개발을 주도했습니다.

E-MAS2는 초당 1억 개 이상의 실시간 데이터 스트림을 처리하기 위해 분산 컴퓨팅 아키텍처를 도입하고, 데이터 병렬 처리 기술을 적용했습니다. 또한, 모든 데이터를 차트에 직접 바인딩하는 대신, 줌/팬 기능과 적분 기반 데이터 집계 기능을 갖춘 Virtual Gantt Chart를 개발하여 시각적인 분석 효율성을 극대화했습니다. 이를 통해 사용자는 방대한 데이터를 직관적으로 파악하고, 세부 분석이 필요한 영역을 신속하게 식별할 수 있게 되었습니다.

E-MAS2 개발은 빅데이터 처리, 실시간 분석, UI/UX 최적화 등 다양한 기술적 과제를 해결해야 하는 도전적인 프로젝트였습니다. 하지만 저는 끊임없는 연구와 혁신을 통해 시스템을 성공적으로 구축했고, 삼성 디스플레이의 생산 효율성 향상에 기여했습니다.

한국 시티은행 모바일 앱은 국내 기술 기반 생체 인증 및 비대면 서비스를 네이티브 기능으로 제공하고 있었습니다. 이로 인해 리전 팀에서 주도하는 Angular 기반 Odyssey 프레임워크의 기술적 통합에 어려움을 겪고 있었습니다. 특히, Odyssey 엔진이 Apache Cordova를 통해 네이티브 기능과 연동되는 구조는 기존 앱의 네이티브 서비스와 호환되지 않는 문제를 야기했습니다.

2년 이상 해결되지 못한 이 문제를 해결하기 위해 저는 Angular (Odyssey)와 자체 개발한 네이티브 인터페이스 기술을 통합하는 솔루션을 제시했습니다. 이 솔루션은 기존 앱의 생체 인증 및 비대면 서비스 기능을 유지하면서 Odyssey 프레임워크의 장점을 활용할 수 있도록 설계되었습니다.

KB FCC 프로젝트 - ONE 챗봇

KB금융그룹 통합 챗봇 플랫폼 구축 프로젝트(KB FCC ONE 챗봇)는 KB지주사들의 다양한 업무를 단일 챗봇 시스템으로 통합하는 것을 목표로 했습니다. 하지만 1년 이상 진행된 프로젝트는 KB국민은행 앱의 기술적 문제, 계열사 간 시스템 이질성 등으로 인해 난항을 겪고 있었습니다.

LG CNS는 프로젝트 정상화를 위해 저에게 기술 자문을 요청했습니다. 저는 1년간 누적된 Java(Maven) 기반 레거시 코드를 전면 재검토하고, Multi-Module 아키텍처를 도입하여 시스템 모듈화 및 확장성을 확보했습니다. 또한, 각 계열사 시스템과의 효율적인 연동을 위해 API Gateway를 구축하고 표준화된 데이터 인터페이스를 정의했습니다.

이러한 전략적 접근과 신속한 개발 프로세스를 통해 불과 4주 만에 ONE 챗봇 시스템을 성공적으로 재구축했습니다. 새롭게 구축된 시스템은 안정적인 서비스 운영과 높은 사용자 만족도를 기록하며 KB금융그룹의 핵심 챗봇 플랫폼으로 자리매김했습니다. 이 프로젝트는 레거시 시스템 개선, 모듈화 및 확장성 확보, 이기종 시스템 간 통합 등 다양한 기술적 과제를 해결해야 하는 복잡한 프로젝트였습니다. 하지만 저는 탁월한 문제 해결 능력과 아키텍처 설계 역량을 발휘하여 프로젝트를 성공적으로 이끌 수 있었습니다.

여러 매거진 (모바일, 데스크, 통합 맴버쉽)을 MVVM 패턴으로 한번에 완성

광고 기반 서비스 플랫폼 개발 시, 반응형 웹 기술은 모바일 환경 최적화에 유리하지만 데스크톱의 다양한 광고 배치 및 UX 요구사항을 충족시키기 어려워 별도 개발이 필요한 경우가 많습니다. 허스트중앙의 다수 매거진 사이트 통합 개발 프로젝트 역시 이러한 문제에 직면했습니다. 짧은 개발 기간 내에 4개 매거진의 모바일/데스크톱 버전(총 8개 사이트)을 구축하고, 공통 로그인 및 OK 캐쉬백 연동 등 복잡한 요구사항을 충족해야 했습니다.

이에 저는 MVVM(Model-View-ViewModel) 디자인 패턴을 적용하여 효율적인 개발 및 관리 전략을 수립했습니다. 단일 ViewModel을 통해 다수의 View를 관리하는 1:N 구조를 구현함으로써 코드 중복을 최소화하고 유지보수 용이성을 확보했습니다. 특히, WPF(Windows Presentation Foundation) 개발 경험을 바탕으로 MVVM 패턴에 대한 깊이 있는 이해를 통해 웹 개발 환경에 맞게 디자인 패턴을 적용하여 생산성을 극대화했습니다.

이러한 접근 방식은 짧은 개발 기간 내에 8개 사이트를 안정적으로 구축하고, 효율적인 배포 및 운영 환경을 구현하는 데 크게 기여했습니다. 특히, MVVM 패턴 적용을 통해 코드 재사용성을 높이고, 향후 기능 확장 및 변경에 유연하게 대응할 수 있는 기반을 마련했습니다. 이는 MVVM 디자인 패턴의 높은 활용성과 다양한 기술 스택에 대한 깊이 있는 이해를 바탕으로 복잡한 프로젝트를 성공적으로 이끌어낸 경험입니다.

APM : 모니터링 서비스

한 기업의 연구 과제 프로젝트로 1년 동안 진행되었으나 기존 개발팀의 이탈로 인해 프리랜서 개발자로써 단독으로 설계도 없는 4개월이라는 촉박한 기간 내에 결과물을 완성해야 하는 위기에 처했습니다. 더욱이 고객사는 명확한 요구사항 정의 없이 기술 및 디자인 측면에서 추상적인 기대치만을 제시하여 어려움이 가중되었습니다.

3D : SVG 기반 WebGL 엔진 개발

중국 교육 기관의 학습용 소프트웨어 개발 프로젝트에 참여하여, 복잡한 데이터 관계 시각화 문제를 해결했습니다. 초기에는 SVG 기반 2D 시각화를 시도했으나, 데이터 중첩으로 인해 정보 전달력이 저하되는 문제가 발생했습니다.

이에 3차원 시각화를 통해 가시성을 높이는 방안을 제안했습니다. 기존 2D SVG 결과물에 유클리드 기하학을 접목하고, 3차원 회전 행렬 및 투영 변환 수식을 적용하여 데이터를 3차원 공간에 배치했습니다. 이를 통해 사용자는 다양한 각도에서 데이터를 관찰하고, 복잡한 관계를 직관적으로 파악할 수 있게 되었습니다.

프로젝트 수행 과정에서 3차원 기하학 및 컴퓨터 그래픽스 기술에 대한 깊이 있는 이해를 얻을 수 있었습니다.
